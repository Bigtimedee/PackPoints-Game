Update the “Card-Completion Reward Curve” to be a DAILY CAP (UTC day) with a maximum of 15,000 PackPTS per day for completing up to 200 correct cards (e.g., 10 matches × 20 cards = 200). This daily relationship must be enforced everywhere in PackPTS.

DAILY INVARIANT
For each user, per UTC day:
- C_day = number of correctly completed unique cards that day (max counted = 200)
- P_day_max = 15,000 PackPTS
- The user’s BASE gameplay earnings from correct cards that day must satisfy:
  P_day(C_day) = floor( 15000 * min(C_day, 200) / 200 )
- The system awards incrementally using delta:
  delta = P_day(C_new) - P_day(C_old), where C_new = min(C_old+1,200)

This ensures:
- Max base earnings per day = 15,000
- Any combination of matches/cards that reaches 200 correct cards yields the max
- No overshoot due to rounding or concurrency

A) GLOBAL CONFIG (SINGLE SOURCE OF TRUTH)

Create/Update:
src/config/rewards.ts

export const DAILY_GAMEPLAY_BASE = {
  CARDS_MAX_PER_DAY: 200,
  PTS_MAX_PER_DAY: 15000,
  dayKeyUTC: (d = new Date()) => {
    // YYYY-MM-DD in UTC
    const yyyy = d.getUTCFullYear();
    const mm = String(d.getUTCMonth()+1).padStart(2,'0');
    const dd = String(d.getUTCDate()).padStart(2,'0');
    return `${yyyy}-${mm}-${dd}`;
  },
  pointsForCardsCompleted: (c: number) =>
    Math.floor(15000 * Math.min(c, 200) / 200)
};

  matchId?: string;
  cardId: string;
  now?: Date;

  const dayKey = DAILY_GAMEPLAY_BASE.dayKeyUTC(params.now ?? new Date());

  // Use a DB transaction with SELECT ... FOR UPDATE for counters
  // Steps:
  // 1) Idempotency insert: try insert gameplay_card_daily_events (userId, dayKey, cardId, is_correct=true)
  //    - if conflict (already exists), return { deltaPts: 0 }
  // 2) Lock counters row:
  //    SELECT * FROM user_gameplay_daily_counters WHERE user_id=? AND day_key=? FOR UPDATE
  //    If missing, insert initial row (0,0) then re-select FOR UPDATE.
  // 3) Compute delta:
  //    oldC = counters.cards_completed
  //    newC = min(oldC + 1, 200)
  //    oldP = counters.base_pts_awarded
  //    targetP = pointsForCardsCompleted(newC)
  //    delta = max(0, targetP - oldP)
  // 4) Update counters:
  //    cards_completed = newC
  //    base_pts_awarded = oldP + delta
  // 5) If delta>0, insert wallet_ledger entry EARN_GAMEPLAY_BASE_DAILY amount=delta
  // 6) Commit and return delta + new totals

  // IMPORTANT:
  // - If is_correct=false, do not insert idempotency row or award points.
  // - This function must be the ONLY place base daily points are awarded.
}
