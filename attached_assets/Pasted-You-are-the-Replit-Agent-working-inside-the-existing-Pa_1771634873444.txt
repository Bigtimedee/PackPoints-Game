=====================================================================

- asset_type: SCORE_CARD, DAILY5_RANK_CARD, STREAK_BADGE, LEADERBOARD_SPOTLIGHT

- Output: 1080x1920 PNG “score card” image suitable for TikTok/IG story
- Use HTML-to-image (playwright) OR canvas in Node OR a server-side React render:
  - Keep it simple: generate a styled PNG with username, score, streak, CTA “PackPTS.com”
- Ensure safe-area padding and readable typography.

3) Generate these assets automatically on:
- MATCH_FINISHED
- DAILY5_FINISHED
- STREAK_UPDATED (only when streak hits milestones: 3, 7, 14, 30)

4) Store the image in:
public/generated/share/{date}/{assetId}.png
